## Descripción de la práctica

Este proyecto es un sitio web de portfolio personal desarrollado como práctica para aprender y profundizar en el uso de **Flexbox** para layouts CSS. El sitio muestra información sobre formación académica, experiencia laboral, conocimientos técnicos, prácticas realizadas y habilidades blandas.

La estructura del portfolio incluye:
- Barra lateral con información de perfil (foto, nombre, título y enlaces)
- Secciones de contenido: Perfil, Formación Académica, Experiencia, Conocimiento Técnico, Prácticas y Ejercicios, Habilidades
- Pie de página con enlaces adicionales

## Tecnologías utilizadas

- **HTML5** - Estructura semántica del sitio
- **CSS3** - Estilos y layouts con Flexbox
- **Variables CSS** - Para el sistema de theming
- **Google Fonts** - Fuente Fira Code

## Cómo ver el proyecto

Para visualizar el sitio web, puedes abrir el archivo `index.html` directamente en un navegador o usar un servidor local:

```bash
# Python 3
python3 -m http.server 8000

# PHP
php -S localhost:8000
```

Luego abre `http://localhost:8000` en tu navegador.
